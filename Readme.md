## Project Overview
This repository hosts my capstone project for the Open Source Software course at VITyarthi. I've conducted a structured audit of **Python**, exploring its history, the Python Software Foundation (PSF) license, and its deep integration within the Linux ecosystem. 

As a student focusing on Cloud Computing, I chose Python because it is the backbone of modern automation, cloud orchestration, and backend infrastructure. This project isn't just about code; it's about understanding the "Four Freedoms" and how community-driven tools shape the professional world.

## Project Objectives
1. **Philosophical Audit:** Analyzing the transition of software from proprietary barriers to open collaboration.
2. **Technical Footprint:** Mapping where Python lives in a Linux environment (directories, binaries, and logs).
3. **Automation:** Creating functional shell scripts to handle system reporting, package inspection, and security auditing.
4. **Ethics:** Reflecting on the "shoulders of giants" philosophy that allows modern tech companies to build on free labor.

## Repository Structure
The repository is organized into five functional scripts, each mapping to specific units of the OSS course:

* `script1_system_identity.sh`: Generates a system-wide identity and license report.
* `script2_package_inspector.sh`: Validates the installation and "philosophy" of the target package.
* `script3_disk_auditor.sh`: Audits system directories for permissions and disk usage.
* `script4_log_analyzer.sh`: A robust log scanner with error-handling and retry logic.
* `script5_manifesto_generator.sh`: An interactive tool to generate a personalized OSS philosophy statement.
